# AWS Nextcloud Solution Architecture

This project documentation contains a solution architecture design for deploying Nextcloud on AWS. 

The design focuses on a secure and cost-effective deployment intended for a medium-scale organization (~100 active users).

The architecture is developed as part of the **AWS Academy Cloud Architecting** course and follows AWS best practices and Well-Architected Framework principles.
